#### 2.- Markdown Inside *&lt;html&gt;* Blocks 
Adding *markdown="1"* attr
```
<div markdown="1">
This is **true** [markdown][id] text.
</div>
```
Adding *markdown="1"* attr to a *&lt;td&gt;*
```
<table>
<tr>
<td markdown="1">This is *true* markdown text.</td>
</tr>
</table>
```
