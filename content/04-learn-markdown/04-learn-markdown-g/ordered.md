#### 7.- Lists
Ordered lists *&lt;ol&gt;*
```
1. item 1
2. item 2
renders as ->
<ol>
    <li>item 1</li>
    <li>item 2</li>
</ol>
```
Nestings
```
1. item 1                  <ol><li>item 1</li>
  - item 1.a                   <ul><li>item 1.a</li>
  - item 1.b       =>              <li>item 1.b</li></ul>
2. item 2                       <li>item 1</li>
                           </ol>
```
